INSTRUCCIONES PARA PROBAR EL PROYECTO

1. Coloca la carpeta "LoginSQLiteAndresMunoz" en C:/xampp/htdocs
2. Abre XAMPP y enciende el módulo "Apache" (NO necesitas MySQL)
3. Abre el navegador y entra a: http://localhost/LoginSQLiteAndresMunoz/
4. Prueba registrarte y luego iniciar sesión.
5. Puedes tomar capturas como evidencia.

Base de datos usada: SQLite (no requiere contraseña ni phpMyAdmin)
